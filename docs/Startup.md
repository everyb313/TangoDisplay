# Startup

TangoDisplay can be configured to launch in different ways depending on how you prefer to work — either fully laid out across two windows or with the Live Display tucked away until you need it. Configure this in **Settings › Startup**.

---

## Startup Mode

Two modes are available:

| Mode | Behaviour at launch |
|---|---|
| **Full Experience** (default) | Both the Live Display window and the Settings window open. The Settings window is brought to the front so you can configure things before moving the Live Display to its monitor. |
| **Player Focused** | The Settings window is focused at launch and the Live Display starts **minimised to the Dock**. The setlist tab is selected automatically. Restore the Live Display any time from the menu bar icon or by clicking it in the Dock. |

Use **Player Focused** when you're DJing on a single screen and want the setlist and player controls front-and-centre at launch — the Live Display is one click away when you want it.

---

## Hide Left Menu Bar on Startup

A separate toggle: when on, the Settings window's **left sidebar starts collapsed** at launch. You can expand it again at any time with the standard sidebar control — this only affects the initial state when TangoDisplay opens.

Useful if you mostly work in a single pane (e.g. Setlist) and prefer a wider working area on first launch.

---

## How It Works

- Both settings persist across restarts.
- Both settings are applied once, on the first appearance of the Settings window after launch — subsequent window opens during the same session use the normal "ensure open" behaviour.
- "Player Focused" miniaturises the Live Display once it's been registered by the system; it does **not** prevent it from opening.
